Bom pessoal, tudo bem? Esse Portfólio demonstrado, eu criei com meus conhecimentos em HTML, CSS e JavaScript.

Bom, é uma página simples para servir como currículo. Ela possui navegação suave entre as seções. animações e um design meio dark com azul(escolha pessoal)..

As seções são compostas por:

- início -- apresentação sobre mim
- sobre -- estatísticas(projetos, certificados e um toque de dedicação)
- certificados -- são meus certificados obtidos, com data e o link de visualização
- contato -- são os meios de contato para chegar até a mim, dentre eles estão; WhatsApp, email, LinkedIn e até mesmo o próprio GitHub.

É um projeto puramente Front-End. Vou deixar um mini mapa de como ficou a Estrutura dele:
